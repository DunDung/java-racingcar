# java-racingcar
자동차 경주 게임 미션 저장소

### 기능 목록
* 자동차 이름을 입력 받는다.
    * 자동차 이름은 쉼표를 기준으로 구분한다.
    * 자동차 이름의 길이는 1이상 5이하 이다.
    
* 게임 시도 횟수를 입력 받는다.
    * 시도 횟수는 양수다.
    * 시도 횟수는 정수다.
    
* 각 자동차의 전진 여부를 정한다.
    * 0에서 9사이의 랜덤 값을 생성한다.
    * 해당 범위가 아닐 때 예외 처리한다.
    * 4 이상이면 전진하고 3 이하면 멈춘다.
    
* 자동차 경주의 진행상황을 출력한다.

* 자동차들 중에 우승자를 찾는다
    * 가장 앞서나간 자동차의 Max position을 찾는다.
    * Max position과 같은 위치의 자동차들을 찾는다.

* 우승자를 출력한다.
